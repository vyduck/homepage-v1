Vel | Discord bot

I created this bot in the first week of July, 2024 with inspiration from a Reddit user.

It tracks how much time you spend in a voice channel. This data can then be used to generate leaderboards. The bot also allows users to group channels, for example, there might be a productivity group, a gaming group, or just a chatting group. Leaderboards can also be generated using these groups.

The bot is written in JavaScript and uses NodeJS runtime. It stores information using MongoDB. It communicates with Discord using the discord.js module.

Links: Invite bot. Top.gg page.

Tally | Discord bot

I created this bot in Feb, 2024. And then forgot about it due to exams. When my exams ended in June, 2024, I started to improve the bot, but realized it would be better to rewrite it. So, I re-wrote the bot in the last two weeks of June 2024 with some new features.

Tally can keep count of anything that could be counted in numbers. Kills in a game, total steps walked, total words written, etc.

Previously the bot stored information on local file system. I upgraded the bot to store the information in an online MongoDB instance at the end of October '24.

The bot is written in JavaScript and uses NodeJS runtime. It stores information using MongoDB. It communicates with Discord using the discord.js module.
