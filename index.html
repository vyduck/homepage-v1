Vel | Discord bot

I created this bot in the first week of July, 2024 with inspiration from a Reddit user. It tracks how much time you spend in a voice channel. This data can then be used to generate leaderboards. The bot also allows users to group channels, for example, there might be a productivity group, a gaming group, or just a chatting group. Leaderboards can also be generated using these groups.

The bot is written in Javascript and uses NodeJS runtime. It communicates with Discord using the discord.js module.
